¿que es una base de datos?


Una base de datos es un sistema digital que organiza, almacena y gestiona una gran cantidad de información estructurada en tablas (filas y columnas) para permitir su rápido acceso, modificación y actualización.



¿para que sirve un BD?


Una base de datos (BD) sirve para almacenar, organizar, gestionar y recuperar grandes volúmenes de información de forma estructurada, rápida y segura.


que es un campo


Un campo en una base de datos es cada dato o característica específica que se guarda sobre algo


que es un registro

Un registro en una base de datos es una fila completa en una tabla, que contiene valores para cada campo definido.


que es un dato


Un dato es una unidad básica de información que puede ser almacenada, procesada y analizada en una base de datos.



que es un diagrama entidad relación

Un diagrama entidad-relación (DER) es una representación gráfica que muestra las entidades, sus atributos y las relaciones entre ellas en una base de datos.
